# Project Brainstorm V1 — Single‑Page Replit Build Pack

This pack is designed for **one simple web page app** (a single main screen) built with a tiny server behind it.

## What you do (step by step)
1) In Replit, create a new project using **Python**.
2) Upload this entire folder into the root of the project.
3) Open `PROMPT_SEQUENCE_SINGLE_PAGE.md`.
4) Copy/paste prompts into Replit’s coding agent **one at a time**, in order.
5) After each prompt finishes, click **Run** and check the app works.

## What “single‑page app” means here
- The user mostly stays on **one main page**.
- That page switches between sections: Login / Pending / Dashboard / New Research / Study Viewer / Admin.
- We still use a server (Flask) and a small database (SQLite) so data can be saved.
